## Configuration

Set your environment
Options: [ dev, prd, qas ]

Options

- set NODE_END=dev
- set NODE_END=prd
- set NODE_END=qas

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
